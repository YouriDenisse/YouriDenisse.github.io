Keuzedeel

Bij keuzedeel konden we kiezen voor Java of game development. Ik heb gekozen voor game development omdat ik het leuk vind om met games bezig te zijn. We hebben ervoor gekozen om een soort subway surfers clone te maken. We doen dit in een groepje van 4. De game engine die we gebruiken is Babylon.js, dit is een JavaScript game engine. We hebben ervoor gekozen om de game in 3D te maken, omdat we dit een stuk leuker vinden dan 2D. We hebben al een aantal dingen werkend gekregen zoals het lopen, springen en het ontwijken van obstakels. Ook hebben we al een score systeem gemaakt. We zijn nu bezig met het toevoegen van power-ups en het verbeteren van de graphics. Het is een uitdagend project, maar we leren er veel van en het is erg leuk om te doen!
