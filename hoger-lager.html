Hoger-Lager

Het project hoger-lager is het eerste project waar we mee begonnen waren, vandaar dat het ook de simpelste is. Het doel was om in je groepje een game te maken waar je dobbelstenen gooit, en dan gokt of de computer hoger of lager gaat gooien. We hebben ervoor gekozen om ook de optie toe te voegen om muntjes in te zetten, win je verdubbel je je inzet, verlies je dan verlies je je inzet. Dit was optioneel maar we vonden het een leuke toevoeging. De game moest opgebouwd worden uit HTML/CSS en JavaScript. Het was een hele prettige opdracht om kennis te maken met zowel de code als het werken met scrum. Ik heb het project afgerond met een 8,4 dus ik zou zeggen dat ik het goed beheers.
